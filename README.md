このプロジェクトでは、オープンソースの量子化学計算ツール「PySCF(Python Simulations of Chemistry Framework)」を利用して、物質の吸収スペクトルを求めるための python の解析コード、及び、前処理として候補物質の作成、後処理として解析結果をグラフ化、レポート作成の手順を提供します。

大きくは下記の項目に分類されます。

1. 量子化学計算
    1) 吸収スペクトルを計算する。
    2) 分子軌道 (HOMO/LUMO)を計算する。
    3) 励起状態分子内プロトン移動 ESIPT (Excited state intramolecular proton transfer)を模して、吸収スペクトルを計算する。 

1. 候補物質の作成（前処理）
    2) 修飾する官能基を探す。データベースより官能基のリストを作成し、それらを SMILES で指定した分子骨格に修飾した候補物質を作成する。
    2) 官能基の組み合わせを探す。複数の有効な官能基を、骨格となる分子の複数箇所に修飾した時の候補物質の吸収スペクトルを比較する。

1. 解析結果の可視化(後処理)
   2) 吸収スペクトルの可視化、レポートを作成する。
   3) 分子軌道を可視化する。
